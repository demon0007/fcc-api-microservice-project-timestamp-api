FCC: Timestamp API

This is an FCC project. It is an API which responds with a JSON of the UNIX and UTC format of the date string provided in the URL.

  • This API works on the url /api/timestamp/<date string>
  • <date string> should be in either format:
    • YYYY-MM-DD
    • Unix based Epoch Number
  • If The <date string> is empty it will return with current time as <date string> parameter
  • If <date string> is not in valid format it will return a JSON with error of "Invalid Date"
